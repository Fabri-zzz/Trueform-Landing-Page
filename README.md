# TrueForm

## Descripción
TrueForm es una plataforma fitness impulsada por IA que ayuda a los usuarios a obtener rutinas personalizadas, seguimiento de progreso y recomendaciones adaptadas a sus objetivos.

## Integrantes
- Alvaro Jordan
- Fabricio Rivera
- Lesly Balta
- Marco Huamancayo
- Misael Mejia

## Segmentos Objetivo
- Principiantes en entrenamiento físico.
- Usuarios intermedios.
- Personas que desean mejorar sus hábitos saludables.

## Características
- Rutinas personalizadas.
- Seguimiento de progreso.
- Recomendaciones con IA.
- Historial de entrenamientos.
